# KeyTone Web

Next.js 15 frontend for KeyTone.

## Features

- Landing page with headline/upload/polished sections
- Login/signup with Supabase auth
- Pricing page with Stripe upgrade action
- Protected dashboard with drag-and-drop audio upload
- Upload history with project cards, BPM/key/chords, exports and audio preview
- Credit display and upload block when exhausted

## Setup

1. Install dependencies:

```bash
npm install
```

2. Copy `.env.example` to `.env.local` and configure values.

3. Run:

```bash
npm run dev
```
